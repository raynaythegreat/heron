# Heron Brand Guidelines

## Brand Overview

Heron is the centralized platform for AI-powered business operations. We empower organizations to harness artificial intelligence through a unified, professional hub.

---

## Brand Voice & Tone

### Core Attributes

| Attribute | Description |
|-----------|-------------|
| **Professional** | Authoritative yet approachable; we speak with expertise, not jargon |
| **Accessible** | Complex AI concepts made clear; inclusive language for all skill levels |
| **Innovative** | Forward-thinking and progressive; we lead, not follow |

### Voice Guidelines

**Do:**
- Use active voice and direct language
- Lead with value and outcomes
- Balance technical accuracy with clarity
- Use "we" for company, "you" for customers

**Don't:**
- Use excessive buzzwords or hype
- Over-simplify to the point of inaccuracy
- Sound robotic or impersonal
- Make unsubstantiated claims

### Tone by Context

| Context | Tone |
|---------|------|
| Documentation | Clear, instructive, helpful |
| Marketing | Confident, inspiring, solution-focused |
| Support | Empathetic, patient, thorough |
| Social | Engaging, knowledgeable, approachable |

## Typography

### Font Stack

**Primary:** Inter (sans-serif)
**Secondary:** JetBrains Mono (monospace, for code)

### Type Scale

| Name | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Display | 72px | 700 | 1.1 | Hero headlines |
| H1 | 48px | 700 | 1.2 | Page titles |
| H2 | 36px | 600 | 1.3 | Section headers |
| H3 | 24px | 600 | 1.4 | Subsections |
| H4 | 20px | 600 | 1.4 | Card titles |
| Body Large | 18px | 400 | 1.6 | Lead paragraphs |
| Body | 16px | 400 | 1.6 | Default text |
| Body Small | 14px | 400 | 1.5 | Captions, metadata |
| Micro | 12px | 500 | 1.4 | Labels, tags |

### Typography Guidelines

- Use sentence case for headings (except proper nouns)
- Maximum line length: 72 characters for readability
- Maintain consistent vertical rhythm with 8px base unit
- Never use font weights below 400

## Logo Usage

### Clear Space

Maintain minimum clear space equal to the height of the "H" in "HQ" around all sides of the logo.

### Minimum Sizes

| Format | Minimum Width |
|--------|---------------|
| Digital | 120px |
| Print | 1 inch (25.4mm) |
| Favicon | 16x16px |

### Color Variations

| Version | Usage |
|---------|-------|
| Full color | Default, light backgrounds |
| Inverted | Dark backgrounds |
| Monochrome | Single-color applications |
| Outline | Subtle applications |

### Logo Don'ts

- Don't stretch or distort
- Don't rotate the logo
- Don't add effects (shadows, glows, etc.)
- Don't change colors outside brand palette
- Don't place on busy backgrounds
- Don't alter typography

---

## Iconography

- Use consistent 24x24px grid for UI icons
- 2px stroke weight for outlined icons
- Rounded caps and joins
- Maintain optical alignment

---

## Writing Style

### Grammar & Mechanics

- Use Oxford comma
- Spell out numbers one through nine
- Use title case for headings
- Use sentence case for buttons and labels
- Avoid abbreviations in formal copy

### Key Terminology

| Term | Usage |
|------|-------|
| Heron | Full product name (first mention) |
| HQ | Acceptable abbreviation after first use |
| AI-powered | Hyphenated as adjective |
| Business intelligence | Two words |

---

## Accessibility

- Maintain 4.5:1 contrast ratio for normal text
- Maintain 3:1 contrast ratio for large text (18px+)
- Never rely on color alone to convey information
- Provide text alternatives for all non-text content
- Ensure keyboard navigability for all interactive elements
